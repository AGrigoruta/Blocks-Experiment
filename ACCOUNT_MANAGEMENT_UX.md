## Accessibility Considerations

### Keyboard Navigation
- All OAuth buttons: `Tab` to focus, `Enter` to activate
- Guest mode: `Tab` to focus input, `Enter` to submit
- Settings: Arrow keys for navigation, `Esc` to close
- Logout: `Shift+L` keyboard shortcut

### Screen Reader Labels
- "Sign in with Google button"
- "Sign in with GitHub button"
- "Play as guest button"
- "User menu, signed in as John Doe"
- "Settings button"
- "Logout button"

### Color Contrast
- All text meets WCAG AA standards (4.5:1 contrast ratio)
- Focus indicators clearly visible
- Status messages use icons + text (not color alone)

### Error Messages
- Clear, descriptive error messages
- Actionable instructions
- Multiple recovery options
